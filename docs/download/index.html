Installation

To use the latest version of Mockingbird on your website, simply paste this link in to the <head></head> section of your site: 

<link rel="stylesheet" href="https://unpkg.com/mb-sass@0.9.5-alpha/mockingbird.css">

Alternatively, you can take a link from the list below.

You are a developer who wants to compile his own Mockingbird? Include it in your project via npm:

npm install mockingbird-sass [--save | --save-dev]

https://www.npmjs.com/package/mockingbird-sass
