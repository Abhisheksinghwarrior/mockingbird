Mockingbird

A front-end framework for responsive layouts and easy web development created with SASS

View on GitHub Recommeded Download v0.9.1

Note

The Mockingbird Project is still Work in Progress.
This is a development pre-release. Anything in this package may change or be deleted.

If you experience any bugs or if you want to help improve the framework, then feel free to open an issue on Github.

Installation

To use the latest version of Mockingbird on your website, simply paste this link in to the <head></head> section of your site: 


<link rel="stylesheet" href="https://mockingbird-cdn.surge.sh/mockingbird-latest.min.css">

Alternatively, you can go to the download page for other links.
